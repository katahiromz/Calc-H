概要
--------

ひらがなで入力した計算式を計算できる電卓です。

起動したら、キーボードの[半角／全角]キーを押して
ひらがなを入力できる状態にしてください。

後は、ひらがなで計算式を入力して
[Enter]を押してみてください。結果が表示されます。


動作環境
--------

 * Windows XP/Vista/7/8.1/10
 * 音声合成には、SAPI 5.1以降とSAPIの日本語パックが必要です。


できること
----------

 * 無量大数までの足し算、引き算、掛け算、割り算
 * 分数計算と小数計算
 * パーセント計算と割引・割り増し
 * 余り
 * べき乗
 * 三角関数と双曲関数
 * ルート計算と平方根
 * 指数関数と対数
 * 階乗
 * 逆数、絶対値、足し算の足す数、足される数などを教えてくれる
 * 複数の数の、最大値、最小値、平均値などを教えてくれる
 * 簡単な質問に答えられる

入力例はファイルExamples.txtをご覧下さい。
